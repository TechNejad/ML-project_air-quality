Abstract

Beijing Air Quality Forecast is a course project that analyzes Beijing’s air quality from 2010–2014. The goal is to predict hourly PM 2.5 concentrations and explore how they relate to seasonal and meteorological factors. Using the CRISP–DM methodology, we cleaned and pre‑processed the data, performed exploratory analysis, engineered lagged and rolling features, and trained several regression models. The best‑performing model powers a live web app that lets anyone forecast upcoming air quality and visualize pollution trends in real time.

Key Features

  • Comprehensive Data Pipeline. Handles missing values and outliers, uncovers temporal and meteorological patterns, and creates lagged, rolling window and volatility features to enrich the dataset.
  • Robust Forecasting Models. Trains and compares Random Forest, XGBoost, SVR and KNN regression models. Hyperparameter tuning via GridSearchCV and time series splitting ensures accurate PM 2.5 predictions.
  • Interactive Web Application. A Streamlit app powered by the best‑performing model provides real‑time air quality forecasts with an intuitive interface, allowing users to see whether outdoor air is safe.
